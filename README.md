# Ubuntu Linux Documentation Project

This repository contains step‑by‑step guides for downloading, installing, and configuring Ubuntu Linux on various platforms.

- **Official download sources** – URLs for ISO images.
- **Installation on Windows, macOS** – both as a primary OS and inside Oracle VirtualBox.
- **Linux command reference** – essential commands categorized for quick lookup.
- **Video resources** – curated tutorial videos.

The documentation is organized into logical folders under `Ubuntu_Linux/`.
